# Cam Ring Edge Detection

## 📌 Project Overview
This project performs edge detection on cam ring images using computer vision techniques.  
The system detects object boundaries clearly for analysis and visualization.

## 🎯 Objective
To apply edge detection algorithms on a cam ring image to highlight structural features and boundaries.

## 🛠 Technologies Used
- Python
- OpenCV
- NumPy
- Matplotlib (if used)

## 📂 Project Structure
Cam-Ring-Edge-Detection/
│── Cam_Ring.ipynb  
│── input.jpeg  
│── README.md  

## ⚙️ Working Principle
1. Load input image
2. Convert to grayscale
3. Apply Gaussian Blur (noise reduction)
4. Apply Canny Edge Detection
5. Display detected edges

## ▶️ How to Run
1. Install required libraries:
   pip install opencv-python numpy matplotlib

2. Open the Jupyter Notebook:
   Cam_Ring.ipynb

3. Run all cells.

## 📷 Sample Input
Input image: input.jpeg

## 📌 Output
The output shows detected edges of the cam ring using Canny Edge Detection.
